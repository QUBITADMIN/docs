## 概述

**Qubit Alpha** 是基于 [Hyperliquid](https://hyperliquid.xyz) 的终极资产管理层，提供 AI 驱动的跟单交易和自动化策略。

<CardGroup cols={2}>
  <Card title="跟单者" icon="users">
    自动复制顶级交易员的交易，无需主动管理即可增长您的投资组合。
  </Card>
  <Card title="策略师" icon="crown">
    获取专业级量化基础设施，为您管理的资金提升收益并对冲风险。
  </Card>
</CardGroup>

## 核心使命

我们相信，获取复杂交易策略的渠道不应仅限于专业交易员。Qubit Alpha 通过以下方式实现交易民主化：

<Steps>
  <Step title="发现">
    浏览并分析平台上经过验证的交易员的业绩数据。
  </Step>
  <Step title="连接">
    将您的 Hyperliquid 子账户与您选择的交易员关联。
  </Step>
  <Step title="增长">
    交易被实时自动镜像，坐看您的投资组合增长。
  </Step>
</Steps>

## 核心功能

| 功能 | 描述 |
| --- | --- |
| 🚀 **超低延迟** | 由 Hyperliquid 驱动的 ~100ms 交易执行 |
| 🔒 **非托管** | 您始终保持对资金的完全控制 |
| 📊 **透明分析** | 链上验证的业绩指标 |
| ⚡ **智能风控** | 内置止损和仓位限制 |
